# 🌤️ Weather Forecast App

A modern and responsive weather application built with **HTML5**, **CSS3**, and **Vanilla JavaScript**. The application provides real-time weather information, hourly forecasts, and multi-day forecasts through WeatherAPI, featuring a clean interface, responsive layout, and customizable measurement units.

## ✨ Features

- Search weather by city name
- Real-time weather information
- Current temperature and weather condition
- Hourly forecast
- Multi-day forecast
- Temperature unit conversion (°C / °F)
- Wind speed unit conversion
- Precipitation unit conversion
- Search suggestions
- Loading state
- Error handling
- Responsive design for desktop, tablet, and mobile
- Clean and modern user interface

## ⚙️ Getting Started

Clone the repository:

```bash
git clone https://github.com/Amirh2831/weather-forecast-app.git
```

Navigate to the project folder:

```bash
cd weather-forecast-app
```

Run the project by opening `index.html` in your browser or by using **Live Server** in Visual Studio Code.
